# Weekly OSS Maintainer Report for `example/hello-oss`

## What this project produces

This file is an example of the markdown report generated by `osskit weekly-report`.
A real run includes:

- Open issue triage grouped by bug, feature, docs, question, and needs-triage.
- Changelog draft based on recently merged pull requests.
- Recent failed GitHub Actions runs with links and optional log signals.

## Maintainer workflow

1. Run `osskit weekly-report owner/repo --output report.md`.
2. Review the generated markdown.
3. Copy the useful parts into your release notes, issue comments, or weekly maintainer notes.
